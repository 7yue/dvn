Cross-ref linking

Steps:

1) If all .rst files are in the same SOURCE create an anchor next to the heading or text you want to link to using .. _my-label:  (where my-label is replaced with whatever text you want to use)


2) At the actual link: Indicate the link with the same label as above using
:ref:`Link text <my-label>`
